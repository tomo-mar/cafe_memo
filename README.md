# Cafe Memo App

訪問したカフェの情報を一元管理し、地図上のマッピングと高度な条件検索を可能にする個人的なカフェ記録・データベースアプリです。

## 開発の背景と目的
既存のグルメアプリやメモ帳では、ゲイシャやオンブリゴンといった希少なスペシャルティコーヒーの繊細な特徴や、自分好みの細かなテイスティングノートを柔軟に管理しきれないという課題がありました。
そこで、「自分だけの粒度でカフェ情報を蓄積し、出先から現在地周辺のカフェを直感的に探し出せる専用のデータベース」を構築するため、ノーコードツールを用いて本アプリを迅速にプロトタイピング・開発しました。

## 主な機能
### 1. 自動マッピングとマップ検索
* 登録時に入力した「住所」データから、アプリ内のGoogle Mapへ自動でジオコーディング（ピン配置）を実行します。
* マップ画面を開くことで、現在地周辺の登録済みカフェを視覚的に探し出すことができます。

### 2. 詳細な訪問記録のデータベース化
* 以下のデータ構造で情報を登録・管理（CRUD操作）できます。
  * 店舗名 / 来訪日 / 最寄り駅 / 住所 / コメント（テイスティングノート等）/ 現場の画像
  * Google Mapリンク / Instagramリンク

### 3. 高度な検索・フィルタリング
* リスト（一覧）表示画面から、蓄積したデータベースに対して条件検索が可能です。
* 「店舗名」「最寄り駅」「来訪期間（日付の範囲指定）」などの入力情報から動的にフィルタリングを実行し、目的の記録へ素早くアクセスできます。

### 4. 外部アプリ連携
* マップ上のピン、または一覧画面から店舗の詳細ビューを開くことができます。
* 詳細ビューから、1タップで「Google Mapアプリ」や「Instagram公式アカウント」へ直接ジャンプする動線を設計し、UXを向上させています。
